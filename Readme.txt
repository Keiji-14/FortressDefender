【制作使用エンジン】Unity

【制作期間】１ヶ月

【バージョン管理】SourceTree

【ゲームジャンル】防衛ゲーム

【ゲーム概要】
・このゲームは大砲を操作して降ってくるミサイルを撃ち落とすゲームです。

・ミサイルが地面に着いてしまうとゲームオーバーとなりますので、地面に着く前に撃ち落としましょう。

・振ってくるミサイルを全て撃ち落とすとゲームクリアとなります。残っているミサイルの数は右上のウィンドウから確認出来ます。

・また、地面の手前にバリケードがあり、バリケードはミサイルを防いでくれますが、耐久力があります。

・ゲーム開始前に難易度の選択をする必要があり、難易度は以下の五種類があります。
 （EASY、NORMAL、HARD、VERYHARD、EXTREME

・選択された難易度に応じてミサイルの速度や量が変化します。

ーポイントについてー

・ゲームクリア時にポイントが１つ貰えます。

・ポイントはタイトル画面にある強化項目にて使用することができます。

ー強化についてー

・ゲームクリア時に入手したポイントを使って各項目を強化する事ができます。

・弾の拡大を強化する事で、弾のサイズが大きくなりミサイルに当てやすくなります。

・弾速を強化する事で、発射した時の弾速が早くなり再発射までのクールタイムも短縮されます。

・バリケードを強化する事で、バリケードの耐久力が上がりミサイルを防いでくれる回数が増加します。

【操作説明】
・画面を長押しすると長押しした方向に大砲が向きます。

・画面を長押ししつつ、発射したい方向に向けて画面から指を離すと発射します。

・弾を発射すると再発射までのクールタイムが発生しますので、弾の連射は出来ません。


【アピールポイント・苦労した点】
・一つのゲームシーンだけでタイトル画面、ゲーム画面、ヘルプ画面とシーン遷移を挟まずにゲームを遊べるようにしました。

・敵の配置設定をScriptableObjectで設定、管理が出来る様に対応しました。

・Unityのゲームシーン上では問題なく動作しているのに、Android上で動作が遅くなり調整が難しかった。
